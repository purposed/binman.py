# binman

## Usage

### Installing Software
```bash
$ binman install [-v] github.com/purposed/tidy [--version VERSION]
```

### Updating Software
```bash
$ binman update [-v] [PACKAGE_NAME]...
```

### Listing installed software
```
$ binman list [-v]
```

### Uninstalling software
```
$ binman uninstall [PACKAGE_NAME]
```
